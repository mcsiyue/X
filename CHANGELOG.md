# 更新记录

## v2.8.0

- 进一步修复 Windows 任务栏图标显示为 Tk 羽毛的问题：设置外层/root HWND、窗口类图标和 AppUserModelID。

## v2.7.0

- 自动提取改为逐任务入队：每新增一个任务就立即加入后台自动提取队列，不再等待整批添加完成。
- 自动提取队列会按“并发线程”设置启动多个后台提取线程。
- 修复 Windows 任务栏运行图标可能显示 Tk 默认羽毛图标的问题，强制使用 `推特下载器图标.ico`。

## v2.6.0

- 新增“添加队列后自动提取”：添加链接后自动解析标题、画质、直链和文件大小。
- 新增下载总文件大小统计。
- 下载失败任务自动标红。
- 新增“重试失败”按钮和右键菜单项。
- 新增表头排序，支持按下载进度、文件大小等字段排序。

## v2.4.0

- 新增多线程并发下载，界面可设置并发线程数，默认 3 个线程。
- 新增“文件大小”列，下载时自动读取并显示视频大小。
- 新添加的任务现在会显示在任务列表最上方。
- 优化并发下载时的文件命名，避免同名文件冲突。

## v2.3.1

- 修复关闭到托盘后点击托盘图标无法恢复窗口的问题。
- 新增托盘右键菜单：显示窗口、退出。
- 添加自定义程序图标、窗口图标和托盘图标。

## v2.2

- 批量视频下载基础功能。
- 任务列表、进度展示、错误日志、保存目录选择等功能。
